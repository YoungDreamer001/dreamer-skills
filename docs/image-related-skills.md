# 图像相关技能清单

本清单收录 HK-Skills 中与图像、视觉、配图、图表、截图、GIF、演示视觉等「只要有一点关联」的技能，按使用场景分组。

> 生成时间：2026-06-21  
> 筛选范围：`./bin/hk-skill list` 中所有已安装技能，依据 SKILL.md 描述判断。

---

## 1. 图像生成与编辑

| 技能名 | 说明 |
|--------|------|
| `baoyu-article-illustrator` | 分析文章结构，识别需要视觉辅助的位置，生成 Type × Style × Palette 三个维度的中文正文配图。 |
| `baoyu-comic` | 知识漫画创作，支持多种艺术风格与语气，可批量生成教育漫画分镜。 |
| `baoyu-cover-image` | 文章封面图生成，支持 5 个维度（类型、调色板、渲染、文字、情绪）组合，输出电影宽屏 / 方形等比例。 |
| `baoyu-infographic` | 专业信息图生成，内置 21 种布局 × 22 种视觉风格，输出可发布的信息图。 |
| `algorithmic-art` | 使用 p5.js 创建算法艺术、生成艺术、流场等可交互视觉作品。 |
| `canvas-design` | 使用设计理念创建海报、艺术品等静态视觉，输出 PNG / PDF。 |
| `ian-xiaohei-illustrations` | 生成 Ian 风格的中文正文配图，适合文章、博客、流程、隐喻等场景。 |
| `ian-xiaohei-scenes` | 生成「小黑 + 真实物件 + 物理动作 + 留白叙事」的中文配图，支持标准 16:9 与长卷故事图。 |
| `visual-style-ppt` | 使用 Image 2 模型生成风格驱动幻灯片图片，并组装为纯图片 PPTX。 |

## 2. 图像处理与压缩

| 技能名 | 说明 |
|--------|------|
| `baoyu-compress-image` | 将图片压缩为 WebP 或 PNG，自动选择工具，用于优化图片体积。 |

## 3. 图表、图示与架构图

| 技能名 | 说明 |
|--------|------|
| `baoyu-diagram` | 创建专业暗色 SVG 图表，支持架构图、流程图、时序图、思维导图、时间线等。 |
| `architecture-diagram` | 生成独立的 HTML+SVG 架构图，适用于系统、基础设施、云、安全、网络拓扑。 |

## 4. 视觉 / UI / 网页设计

| 技能名 | 说明 |
|--------|------|
| `baoyu-design` | 创建独立 HTML 设计工件：UI 线框图、交互原型、落地页、仪表盘、App 界面、幻灯片、视觉探索。 |
| `web-design-engineer` | 使用 HTML/CSS/JavaScript/React 构建精美网页工件：页面、仪表盘、原型、幻灯片、动画、UI 模型。 |
| `web-design-guidelines` | 审查 UI 代码是否符合 Web Interface Guidelines，覆盖可访问性、UX、设计审计。 |
| `design-an-interface` | 使用并行子代理为模块生成多种截然不同的接口设计方案。 |
| `frontend-design` | 为新 UI 或现有 UI 重塑提供有辨识度的视觉设计指导，包括美学方向、字体排印。 |
| `frontend-ui-engineering` | 构建生产级用户界面，处理组件、布局、状态与视觉质感。 |
| `ce-frontend-design` | 构建有真正设计质量的前端界面，避免 AI slop，适用于落地页、Web App、仪表盘、组件、交互体验。 |
| `brand-guidelines` | 将 Anthropic 官方品牌色与字体应用于需要品牌视觉风格的工件。 |
| `theme-factory` | 为幻灯片、文档、报告、HTML 落地页等工件提供主题样式工具包，含 10 套预设配色与字体。 |

## 5. 演示文稿、卡片与视觉文档

| 技能名 | 说明 |
|--------|------|
